Задание 1 # Создать приложение, которое загружает и отображает случайные фотографии собак.
Создайте компонент DogGallery
Используйте useState для хранения:
Массива URL фотографий
Состояния загрузки (isLoading)
Используйте useEffect для первой загрузки
Используйте axios для запросов к Dog API
API для запросов: https://dog.ceo/api/breeds/image/random
При первом запуске — загружается 3 случайные собаки
Кнопка "Добавить собаку" — добавляет +1 новое фото в массив
Кнопка "Обновить всё" — полностью заменяет все фото на новые
Пока фото грузятся — показывать текст "Загрузка..."
Критерии выполнения
При открытии страницы видно 3 собаки
При нажатии "Добавить собаку" появляется 4-я, потом 5-я и т.д.
При нажатии "Обновить всё" — все фото заменяются на новые
Пока грузятся фото — висит "Загрузка..."
Ошибки обработаны через try/catch
Индикатор загрузки для каждой кнопки — отдельный стейт для добавления и для обновления
Очистить все — кнопка, удаляющая все фото
Количество собак — отображать счетчик: "Загружено собак: X"
